defaultConsoleLogger module
=======

Default logger provider for magpie/log packet.
It uses the console object to log messages.

Log level and console assignments
---------------------------------------

|Log level||
|-|-
|TRACE|console.debug
|DEBUG|console.debug
|INFO|console.info
|WARN|console.warn
|ERROR|console.error
|FATAL|console.error
